1. Project Title & Description

Title: Izumi — Intelligent E-Learning Management Platform

Izumi is a full-stack, web-based e-learning platform that enables educators to create, publish, and monetize structured online courses while providing students with an interactive learning experience. Built with a React + Vite front-end and a Node.js / Express back-end backed by MongoDB, the system supports four distinct user roles: Student, Teacher (Instructor), Reviewer, and Admin.

Teachers author courses composed of hierarchical modules (text lessons, video content, and graded quizzes) and submit them through a formal approval workflow managed by Reviewers. Students browse an approved course catalog, purchase courses via an integrated payment/transaction subsystem, and track per-module progress with weighted grading policies (threshold, weighted-average, or all-pass). A real-time chat system (Socket.IO) allows direct messaging between students and instructors within course contexts. The Admin dashboard provides platform-wide user management, enrollment oversight, and analytics.

Key features include Google OAuth integration, brute-force login protection, file/media uploads, course rating & reviews, enrollment analytics, Swagger-documented REST APIs, and comprehensive role-based access control — making Izumi a production-grade LMS solution.
